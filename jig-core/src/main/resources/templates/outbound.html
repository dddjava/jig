表示要素

永続化操作方法

出力インタフェース（外部へのアクセス）の構造を可視化するドキュメントです。

用語

出力ポート
アプリケーション層で定義される出力インタフェース（例: Repository インタフェース）。「何をするか」を定義し、実装の詳細には依存しません。
アダプタ
出力ポートの実装クラス。外部アクセサを使用する具体的な実装を担います。
外部アクセサ（永続化）
データベースへの直接アクセスを担うクラス（例: MyBatisやSpringDataJDBCのインタフェース）。
外部アクセサ（その他）
データベース以外の外部システムへのアクセスを担うクラス（例: HTTP クライアント、メッセージキュークライアント）。外部型を使用するものは一律こちらになります。
外部操作対象（永続化）
永続化操作の対象となるリソース（例: テーブル名）。SELECT / INSERT / UPDATE / DELETE の種別とともに表示されます。
外部操作対象（外部型）
外部アクセサが呼び出すクラス。解析対象クラスのインスタンスフィールドが解析対象外の場合に「外部型」としています。

ダイアグラムの構造

各要素の関係と配色は以下のとおりです。

  • 出力ポートは他ドキュメントからのリンク先になるので着色しています。
  • 出力ポートとアダプタはサイドバーの表示設定チェックボックスで表示・非表示を切り替えられます。
  • 永続化アクセサ・外部アクセサはデフォルトで非表示です。チェックボックスで表示できます。
