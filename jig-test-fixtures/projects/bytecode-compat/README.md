# bytecode-compat

## 固定する契約

Java 8 / 21 / 25 でコンパイルしたクラスファイルを、同じように解析できること（`docs/adr/java_version_support_policy.md`）。

## 必要な出力

同一ソースを `--release` で切り替えてコンパイルした三つの出力から、同じ型・メンバー・型間の関連が得られること。

## 制約

**Java 8 でコンパイルできる構文だけを使う**。record・var・sealed・テキストブロックなどは使えない。

型の種類（クラス・インタフェース・enum）、フィールド、メソッド、継承・実装、メソッド呼び出しによる型間の関連を一通り含める。解析結果がバージョン間で一致することを見るためのものなので、要素を増やすより最小構成を保つことを優先する。
