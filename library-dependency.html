JIG / ライブラリ依存情報

解析対象パッケージから、解析対象外のライブラリ（外部ライブラリや Java 標準）への依存を俯瞰します。 ライブラリは既知のもの（spring-web、mybatis、junit など）でまとめて表示されます。 既知でないパッケージは先頭から 2 階層で自動集約します。

Java 標準（java.*, javax.*）は既定で非表示です。下のチェックボックスで切り替えられます。

ライブラリはノードまたは下の表の行をクリックして選択できます。選択中はそのライブラリに関連する依存のみが表示されます。
